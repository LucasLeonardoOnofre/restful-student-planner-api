# RESTful Student Planner API

RESTful CRUD API built with **Node.js**, **Express.js**, and **MySQL** for managing student study tasks.

## Features

- Create tasks
- Retrieve all tasks
- Retrieve task by ID
- Filter tasks by status
- Search tasks by title
- Update tasks
- Delete tasks
- Delete all tasks
- Async/await database operations
- JSON responses with HTTP status codes

---

## Technologies

- Node.js
- Express.js
- MySQL
- mysql2
- dotenv
- ESM Module
